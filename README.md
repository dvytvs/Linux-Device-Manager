Overview A hardware monitoring and auditing tool for Linux-based systems. This application provides comprehensive information about system configuration, offering deeper insights compared to standard system utilities.

Key Features:

  1. System Architecture: Detailed specifications of CPU, RAM, and kernel parameters.

  2. Peripherals & Buses: Thorough analysis of connected PCI and USB devices.

  3. Network Subsystem: Monitoring of network interfaces and their respective properties.

  4. Storage Management: In-depth data regarding disk drives and partition structures.

Deployment: The project is available in the following formats:

  1) DEB: Package for Debian-based distributions (Ubuntu, Linux Mint, etc.).

  2) AppImage: Universal standalone binary.

  3) TAR.GZ / TAR.ZST: Compressed archives for manual installation or Arch-based systems.
